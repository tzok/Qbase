Help

ONQUADRO stores information about experimentally determined quadruplex structures from the Protein Data Bank [1]. The data - nucleic acid sequences, 2D and 3D structures, conformational parameters, topology-driven classifications [3], and visualizations - can be explored at the level of tetrads, quadruplexes, G4-helices, or entire molecules containing these motifs. ONQUADRO database and statistics are automatically updated every week. The newsletter informs the subscribers about all newly added structures.

Database contents

ONQUADRO database contains data organized in four groups (tables): Tetrads, Quadruplexes, G4Helices, and Molecule Structures. Each table allows descending and ascending sorting of numerical data fields and alphabetically sorting the string data fields (A-Z or Z-A). The content of the tables are described in the following sections of this document. In this database, users can find sequences, 2D and 3D structures, as well as diverse tetrad and quadruplex representations and classifications. For 2D structures ONQUADRO presents extended dot-bracket notation (called two-line dot-bracket), VARNA [4] , and R-chie [5] representations, and ONZM classification [3]. All 3D structures are available to be displayed using LiteMol Viewer application [6]. Additionally, quadruplexes and G4Helices can be viewed in a layer diagram. 3D structures quadruplexes are also associated with geometric formalism proposed by Webba da Silva group [7, 8, 9] using chi angles and loops presented within a quadruplex. All representations and classifications available in ONQUADRO are described in Representations and Classifications sections, respectively.

Structure representations

2D structure representations

Dot-bracket notation

Dot-bracket notation represents 2D structures using dots and parentheses. Each dot encodes a single unpaired nucleotide and parentheses represent paired nucleotides. Due to crossing pairings in some complex 2D structures (i.a. pseudoknots), the hierarchy of the occurrence of parentheses was established. The order of the brackets‚Äô appearance is as follows:

  1. ( ) - brackets
  2. [ ] - squared brackets
  3. {{ "{" }} } - curly brackets
  4. < > - angle brackets
  5. A a till Z z (ordered alphabetically)

The simple example of hairpin loop sequence folded into a 2D structure and encoded in dot-bracket notation is presented below:

AAGCAAAGCUU

((((...))))

Each nucleotide in the sequence has a corresponding bracket or dot in the latter row of the example. In green highlighting we marked both the first nucleotide of the pair and its corresponding opening bracket. Orange highlighting reflects the second nucleotide of the same pair and its dot-bracket encryption of a closing bracket.


The above example has a limitation, each nucleotide can be connected to a maximum of one other nucleotide. It was necessary to extend the basic dot-bracket notation to enable encoding 2D structure of tetrads. Thus, we proposed adding one more line into the encryption and transforming the one-line dot-bracket notation into a two-line dot-bracket format. First line of the notation is determined by the nucleotide of a tetrad which is the nearest to the 5' end. It is connected with two other nucleotides with its Watson-Crick, Hoogsteen, or Sugar edge. In [3] we set the priority hierarchy of the edges: (1) Watson-Crick, (2) Hoogsteen, (3) Sugar. That indicates, if the first nucleotide of the tetrad is connected with nucleotide X by Watson-Crick edge and with the second one, Y, with either Hoogsteen or Sugar edge, first line of the two-line dot-bracket notation will follow the encoding for the connection with X nucleotide. The connection with Y nucleotide will be encoded in the second line of the notation. With two-line dot-bracket notation each tetrad is encrypted as a cycle of four linkages. Below example shows the 2D structure of quadruplex. Connections with the first nucleotide of a first tetrad are highlighted with green color.

AAAGGGTTAGGGTTAGGGTTAGGGAA

...[([...{{ "{" }})]...((}...]))..

...([{{ "{" }}...)((...))(...)]}..

More information on two-line dot-bracket notation can be found in [3].

Arc diagram (R-chie)

2D structure of a quadruplex can be represented by R-chie diagrams. The diagram shows the same information as the extended dot-bracket notation but in a graphical way. Each arc represents one connection between two nucleotides. Upper arcs are determined by the connection established for the first nucleotide of the first tetrad in a quadruplex (the same mechanism as in two-line dot-bracket format).

Figure 1. R-chie diagram of quadruplex. The diagram shows connections between residues among all tetrads within the quadruplex.

Classic diagram (VARNA)

The VARNA tool was proposed by Yann Ponty‚Äôs group [4]. It was designed for RNA secondary structure automated visualization. VARNA Java Applet draws 2D structures, nucleotides are embedded within circles and displayed in equal distances between the preceding and following nucleotides. All non-canonical interactions are encoded in Leontis/Westhof classification [11] and displayed in structure as pictograms.

Figure 2. VARNA representation of quadruplex. Tetrads in VARNA format are cycles connected with black lines and pictograms reflecting the interactions between the nucleotides.

3D structure models

Full-atom visualisation (LiteMol)

LiteMol Viewer [6] is a tool for 3D molecule visualization in both web browsers and mobile devices. ONQUADRO utilizes this viewer for interactive display of tetrads, quadruplexes, and G4Helices. In these visualizations users can manipulate the structure, the type of representation (balls and sticks, VDW balls, surface, cartoon, c-ùõº trace), colors of the background and the structure. Users can also select atoms to reveal atomic details. The adjusted and personalized view of the 3D molecule image can be directly saved to users' local machine in png format.

Layer diagram (DrawTetrado)

Layer diagram shows the 3D models of quadruplexes and G4Helices in a simplified view. Each tetrad is demonstrated as a planar rhombus with defined each nucleotide and its order number within a sequence.

Classifications

ONQUADRO incorporates two quadruplex classifications: ONZM [3] for 2D structures, and Webba da Silva geometric formalism [7, 8, 9] for 3D structures. Both classifications are described in the following sections.

ONZM

The secondary structure of both tetrads and quadruplexes can be presented as a cyclic graph in which nucleotides of a tetrad(s) are connected via hydrogen-bonding interactions. As mentioned in the dot-bracket notation section, we set the interaction hierarchy where the highest priority have interactions starting from the Watson-Crick edge, then the Hoogsteen edge interactions, and the lowest priority is set up for the connections starting with Sugar edge. Thus, the three possible arrangements of the connections within a tetrad (displayed in a graph) form a shape of O, N, and Z letters. These are the basic classes of ONZ classification [3] of a single tetrad. If we name the first nucleotide (the closest to the 5‚Äô end) of a selected tetrad as N1 and following nucleotides of the same tetrad in 3‚Äô end direction as N2, N3, N4, then we can define ONZ classes as pairings:

  • O - (N1, N2), (N2, N3), (N3, N4), (N4, N1),
  • N - (N1, N2), (N2, N4), (N4, N3), (N3, N1),
  • Z - (N1, N3), (N3, N2), (N2, N4), (N4, N1),

This ONZ arrangement assumes following connections from 5‚Äô end to 3‚Äô end. Let us visualize it as a clockwise progression of the nucleotide connection chain. In such case, the classes are attributed with the + symbol (O+, N+, Z+) to distinguish it from an anticlockwise progression of the nucleotides connection (from N1 to 3‚Äô end direction). The counter direction is attributed with - symbol, giving us another three subcategories of ONZ classes: O-, N-, Z-. All the clockwise and anticlockwise classes are presented in Figure 5.

Figure 5. The ONZ classes divided into six subcategories: O+, O-, N+, N-, Z+, Z-. Arrows show the direction of connections, black circle pictograms denote Watson-Crick edges, square pictograms represent Hoogsteen edges.

For quadruplexes, the ONZ classification needed to extend to ONZM classification. The fundamental rule for a single quadruplex is a structure built of at least two tetrads. If the quadruplex consists of tetrads of mixed classes, e.g. O and Z, it has to be labeled as the M (mixed) type. For the quadruplexes, ONZM classes are given the additional attribute specifying the order of nucleotides within the tract (nucleotides engaged in a quadruplex that are closed to each other and belong to different tetrads). The antiparallel arrangement has the a suffix, the parallel - p suffix, the hybrid - h. Parallel arrangement has tracts directed in the same course. Antiparallel has two tracts directed in course opposite to other two tracts. When at least one tract is opposite from the other three tracts direction - the hybrid direction is established. Thus, in quadruplexes we can distinguish twelve subclasses: Op, Oa, Oh, Np, Na, Nh, Zp, Za, Zh, Mp, Ma, Mh.

Webba da Silva geometric formalism

The formalism proposed by Webba da Silva [7, 8, 9] was created based on glycosidic bond angles (GBA) and loops. The proposed topology concerns unimolecular quadruplexes. Each nucleotide within a tetrad has one of the two conformations of the base/sugar relation ‚Äì syn or anti. Both conformations can be described by the glycosidic torsion angle œá. Syn architecture is present when -90¬∞ ‚â§ œá ‚â§ +90¬∞, anti in condition when +90¬∞ ‚â§ œá ‚â§ +180¬∞. Depending on the conformation of each nucleotide and its neighbours connected with hydrogen bonding, da Silva research group established three types of groove, that appears between the nucleotides: narrow (n), medium (m), and wide (w). This information can be directly translated into quadruplex grooves as in stable quadruplexes tetrads can be stacked only if they have the same groove width.

The research group aligned the groove identification with the loop types. A lateral loop (l) connects nucleobases that share hydrogen bonds (i.e. nucleobases of the same tetrad). A propeller loop (p) links nucleobases that share a groove but are not a part of the same tetrad. A diagonal loop (d) creates linkage between nucleobases that do not share hydrogen bonds but are still part of the same tetrad. Quadruplexes are built of three loops, each of them can be one of the three possible types but not all combinations are possible to exist. Thus, da Silva group presented 26 permissible loop topologies with groove description (all listed in Table 1). Each quadruplex loop descriptor has also established attributes for loop progression. The clockwise progression of a loop (5‚Äô to 3‚Äô) has a + prefix to a loop type, the anticlockwise progression (3‚Äô to 5‚Äô) has a ‚Äì prefix.

Browsing the data

Two search tools are available for users - one in the main page, for searching through articles' information, another one after selecting one of four table views from the home page ‚Äì Tetrads, Quadruplexes, G4Helices, or Structures.

Quick search

Home page search allows to filter structures that contains provided key word(s) in at least one of the following: abstracts, publication titles, authors, journals name, published dates, publication DOIs. Simple logical operators ‚Äì AND, OR, NOT, can be used to narrow the search area. AND operator allows to search for structures with both defined key words, OR ‚Äì searches for structures with either one of the key words, NOT ‚Äì excludes the latter key word from search. Results are sorted based on their accuracy.

Tetrads

Tetrad is a basic building unit of a quadruplex. It is composed of four nucleotides in pseudo-planar architecture. In G-quadruplexes, tetrads are built of guanines, however, tetrads can form also from other residues. Tetrads table allows users to browse all tetrads stored in the ONQUADRO database. Each tetrad has an individual tetrad ID value composed of T prefix and its own sequential number. A single tetrad is always connected with:

  • Quadruplex ID, unique id of a quadruplex related to the tetrad,
  • PDB ID, link to PDB web page of a structure that contains the tetrad,
  • PDB Deposition, date of deposition of the molecule,
  • Assembly ID, the functional form of the molecule,
  • Molecule, type of nucleic acid (one of three: DNA, RNA, or Other),
  • Experimental method, type of used study technique (NMR or X-Ray),
  • Sequence, the sequence of the tetrad,
  • Ion, name of ion present within the tetrad,
  • Ionic charge, number and charge of the element,
  • ONZ class, assigned ONZ class according to [2].
  • Tetrad combination, tetrad classification according to Webba da Silva [7, 8, 9].

Clicking the tetrad id link redirects users to the web page with detailed information of a selected tetrad. Beside the text data, users are provided with dot-bracket notation, VARNA and R-chie representations of a 2D tetrad structure, and interactive application (LiteMol Viewer) to view the 3D model of a tetrad. All the information included on this page, excluding the interactive model, are available for download. This functionality is enabled by the Save data button on the web page.

Quadruplexes

Quadruplexes are built of at least two tetrads. This means each quadruplex in the Quadruplex table is in a 1:N relation with tetrads. Each quadruplex has an individual quadruplex ID value composed of Q prefix and its own sequential number. All stored quadruplexes are connected with the following attributes:

  • PDB ID, link to PDB web page of a structure that contains the tetrad,
  • PDB Deposition, date of deposition of the molecule,
  • Assembly ID, the functional form of the molecule,
  • Molecule, type of nucleic acid (one of three: DNA, RNA, or Other),
  • Experimental method, type of used study technique (NMR or X-Ray),
  • Sequence of tetrads, the sequence of the tetrads, divided by comma,
  • Ion, name of ion present within the quadruplex,
  • Ionic charge, number and charge of the element,
  • Type (by no. of strands), type of the quadruplex based on number of strands (one of three: unimolecular, bimolecular, or tetramolecular),
  • Type (by ONZM), type of the quadruplex determined by differentiation of the ONZ classes assigned to tetrads that are building the quadruplex. Regular quadruplexes contain tetrads of the same ONZ class progression, either clockwise (+) or anticlockwise (-), whereas the irregular quadruplexes have tetrads of both types, clockwise and anticlockwise. For more information, please refer to [10],
  • ONZM class, assigned ONZM class according to [3],
  • No. of tetrads, number of tetrads which create the quadruplex.
  • Loop topology, loop topologies according to Webba da Silva [7, 8, 9],
  • Tetrad combination, tetrad classification according to Webba da Silva [7, 8, 9].

Clicking the quadruplex id link redirects users to the web page with detailed information of a selected quadruplex. Beside the text data, users are provided with dot-bracket notation, VARNA and R-chie representations of a 2D quadruplex structure, and interactive application (LiteMol Viewer) to view the 3D model of a quadruplex. All the information included on this page, excluding the interactive model, are available for download. This functionality is enabled by the Save data button on the web page.

G4Helices

G4-helix is formed by stacking interactions of G-tetrads, which are independent on backbone connectivity [2]). In ONQUADRO, each helix has an individual helix ID value composed of the H prefix and its own sequential number. All stored helices are connected with the PDB ID which are linking the helices to the 3D PDB structure they originate from. Beside the G4Helix ID following attributes are displayed in the table:

  • PDB ID, link to PDB web page of a structure that contains the tetrad,
  • PDB Deposition, date of deposition of the molecule,
  • Assembly ID, the functional form of the molecule,
  • Molecule, type of nucleic acid (one of three: DNA, RNA, or Other),
  • Experimental method, type of used study technique (NMR or X-Ray),
  • Sequence of tetrads, the sequence of the tetrads, divided by comma,
  • Type (by no. of strands), type of the quadruplex based on number of strands (one of three: unimolecular, bimolecular, or tetramolecular),
  • No. of quadruplexes, number of quadruplexes the helix is built of,
  • Quadruplex ID, ids of quadruplexes related to the G4Helix,
  • No. of tetrads, number of tetrads the helix is built of.

Clicking the G4Helix id link redirects users to the web page with detailed information of a selected G4Helix. Beside the text data, users are provided with dot-bracket notation, VARNA and R-chie representations of a 2D G4Helix structure, and interactive application (LiteMol Viewer) to view the 3D model of a G4Helix. All the information included on this page, excluding the interactive model, are available for download. This functionality is enabled by the Save data button on the web page.

Molecule Structures

Highest level of the structure types, each structure is composed of at least one quadruplex motif. Structures are presented as PDB IDs. Each record in Structures table has:

  • PDB Deposition, date of deposition of the molecule,
  • Assembly ID, the functional form of the molecule,
  • Molecule, type of nucleic acid (one of three: DNA, RNA, or Other),
  • Experimental Method, method used for analysing the structure,
  • Quadruplex ID, id of the quadruplexes associated with the molecule,
  • 2D structure, VARNA image of the quadruplex,
  • 3D structure, interactive model of a molecule.

Clicking the PDB id link redirects users to the PDB web page.
